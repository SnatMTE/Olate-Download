# Olate Download (Forked Version)

## Description
This repository is a fork of **Olate Download**, updated to modern coding standards using PHP PDO, improved structure, and current best practices. It preserves the original functionality while making the codebase easier to maintain, extend, and integrate with other projects.

This fork contains patches and updates that were applied for [Amazing World Music](https://www.amazingworldmusic.com). I do not intend to add new features myself, but others are welcome to fork this repository, update it, and contribute improvements.

---

## Features
- Updated PHP code to use PDO instead of deprecated MySQLi methods
- Modern folder and file structure for easier maintenance
- Compatibility improvements for current PHP versions
- Secure handling of user inputs
- Maintains original download and management functionality
- Includes patches made specifically for Amazing World Music

---

## Requirements
- PHP 8.0 or higher
- MySQL or MariaDB database
- Web server such as Apache or Nginx
